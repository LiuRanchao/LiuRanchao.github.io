	> A RecyclerView(advanced and flexible version of ListView in Android) with refreshing,loading more,animation and many other features.

## 二，框架功能

1. [AutoService](https://github.com/google/auto/tree/master/service)

	> AutoService will generate the file META-INF/services/javax.annotation.processing.Processor in the output classes folder.    
	> 用于自定义注解时，自动生成 META-INF/services/javax.annotation.processing.Processor

2. [JavaPoet](https://github.com/square/javapoet)

	> JavaPoet is a Java API for generating .java source files.    
	> 生成.java文件

3. [EventBus](https://github.com/greenrobot/EventBus)

	> Android optimized event bus that simplifies communication between Activities, Fragments, Threads, Services, etc. Less code, better quality

4. [butterknife](https://github.com/JakeWharton/butterknife)

	> Bind Android views and callbacks to fields and methods. http://jakewharton.github.io/butterknife/
	
5. [greenDAO](https://github.com/greenrobot/greenDAO/tree/master/DaoGenerator)

	> greenDAO is a light & fast ORM solution for Android that maps objects to SQLite databases. http://greenrobot.org/greendao/
	
6. [glide](https://github.com/bumptech/glide)

	> An image loading and caching library for Android focused on smooth scrolling
	
7. [Fresco](https://github.com/facebook/fresco)

	> An Android library for managing images and the memory they use. http://frescolib.org/
	
8. [dagger](https://github.com/square/dagger)

	> A fast dependency injector for Android and Java. http://square.github.io/dagger/

9. [retrofit](https://github.com/square/retrofit)

	> Type-safe HTTP client for Android and Java by Square, Inc. http://square.github.io/retrofit/
	
10. [robospice](https://github.com/stephanenicolas/robospice)

	> Repo of the Open Source Android library : RoboSpice. RoboSpice is a modular android library that makes writing asynchronous long running tasks easy. It is specialized in network requests, supports caching and offers REST requests out-of-the box using extension modules.	
11. [moshi](https://github.com/square/moshi)

	> A modern JSON library for Android and Java. http://square.github.io/moshi/1.x/moshi/
	
12. [KLog](https://github.com/ZhaoKaiQiang/KLog)

	> 这是一个Android专用的LogCat工具，主要功能为打印行号、函数调用、Json解析、XML解析、点击跳转、Log信息保存等功能
	
13. [timber](https://github.com/JakeWharton/timber)

	> A logger with a small, extensible API which provides utility on top of Android's normal Log class. http://jakewharton.github.io/timber/

14. [logger](https://github.com/orhanobut/logger)

	> Simple, pretty and powerful logger for android
	
15. [routable-android](https://github.com/clayallsopp/routable-android)

	> Routable, an in-app native URL router, for Android. 启动Activity跳转封装
	
16. [android-saripaar](https://github.com/ragunathjawahar/android-saripaar)

	> UI form validation library for Android
