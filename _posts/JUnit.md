#常用方法：

| 方法 | 描述 | 
| ------ | ------ | 
| assertEquals(boolean expected, boolean actual) | 检查两个变量或者等式是否平衡 | 
| assertFalse(boolean condition) | 检查条件是假的 |
| assertNotNull(Object object) | 检查对象不是空的 |
| assertNull(Object object)  | 检查对象是空的 |
| assertTrue(boolean condition) | 检查条件为真 |
